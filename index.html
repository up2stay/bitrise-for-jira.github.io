Introduction

Welcome to the Jira-Bitrise Integration Plugin Documentation. This plugin is designed to streamline your software development workflow by linking Bitrise builds to Jira issues. By consuming webhook events from Bitrise, this integration provides a seamless way to keep track of your development and issue management in one place. This documentation will guide you through the installation, configuration, and usage of this integration.

Installation

Before you can begin using the Jira-Bitrise Integration, you'll need to install it. Follow these steps to get started: Prerequisites:
1. You must have administrative access to your Jira instance.
2. You should have a Bitrise account with access to the projects and builds you want to link with Jira issues.

To install the plugin:
1. Log in to your Jira instance with administrative privileges.
2. Click on the 'Settings' gear icon in the top right corner and select 'Manage Apps.'
3. Click on 'Find new apps' in the left sidebar.
4. Search for 'Bitrise for Jira' in the search bar.
5. Click on 'Install.'
6. Click on 'Get started.'
7. Follow the instructions on the configuration page to set up the plugin.

Usage

Now that the integration is configured, it will automatically link your Bitrise builds to the specified Jira issues. When a build is completed, a webhook event will be sent to Jira, associating the build with the Jira issue you defined. You can track your Bitrise builds and their associated Jira issues directly from your Jira project.
